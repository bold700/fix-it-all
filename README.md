# Fix-It All — landingspagina (lead demo)

Single-file responsive landingspagina voor autoservice Fix-It All (Utrecht).
Gebouwd als demo om aan de prospect te tonen.

- `index.html` — alles in één bestand (HTML + Tailwind via CDN)
- Mobiel + desktop responsive, focus states, `prefers-reduced-motion`

## Lokaal bekijken
Open `index.html` in de browser, of:
```bash
python3 -m http.server 8099
```
